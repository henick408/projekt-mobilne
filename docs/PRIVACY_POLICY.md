# Polityka prywatności — Katalog Przedmiotów

Data obowiązywania: 2026-06-04

Aplikacja Katalog Przedmiotów umożliwia tworzenie prywatnego katalogu przedmiotów. W celu działania aplikacji przetwarzane są adres e-mail użytkownika, opisy przedmiotów, kategorie oraz zdjęcia wykonane przez użytkownika.

Dane są przechowywane w usługach Firebase Authentication i Cloud Firestore dostarczanych przez Google. Dane katalogu są przypisane do konta użytkownika. Aplikacja nie sprzedaje danych, nie wyświetla reklam i nie udostępnia danych innym użytkownikom.

Aplikacja prosi o dostęp do aparatu wyłącznie w celu wykonania zdjęcia przedmiotu. Plik tymczasowy jest tworzony w prywatnym katalogu cache aplikacji i po zapisaniu danych jest usuwany. Zdjęcie nie jest dodawane do galerii telefonu.

Użytkownik może usunąć wpisy z poziomu aplikacji. W celu usunięcia całego konta i danych należy skontaktować się z administratorem projektu studenckiego.
